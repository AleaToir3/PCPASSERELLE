Yoga Saloon - énoncé

Introduction

Le but de cet exercice va être de s’initier à Flexbox. Une nouvelle façon d’agencer son interface de façon intelligente. C’est aussi l’occasion de découvrir les pseudo-éléments et de travailler les pseudo-classes.

Énoncé

Intégrer la maquette suivante pour un salon de yoga proposant des cours de yoga suivis d’un apéro, plus ou moins healthy 🥤.

Cette fois absolument tout le contenu est fixé à une largeur de 1060px. Il n'y a pas de fonds débordants.

Typographies utilisées

  • Titre h1 : Mandali
  • Raleway

Couleurs

  • Bleu : #6E8E9B

Liens recommandés

Pour cet exercice, vous aurez besoin de :

  • Flexbox
  • Pseudo-classe
  • Pseudo-élément
  • Media Queries

Culture

Niveau CSS, on peut aussi bien utiliser :after que ::after pour cibler un pseudo élément. Pour autant, on va préférer utiliser un : quand il s’agit d’une pseudo-classe et :: quand il s’agit d’un pseudo-élément.
