Cases à cocher

Dans les informations personnelles, on ne doit pouvoir sélectionner qu'un petit rond à la fois. Il faut donc trouver comment faire pour que quand on clique sur "Homme", alors que "Femme" était déjà coché, alors ça le décoche !

Faciliter le remplissage du formulaire

Vous avez peut-être déjà remarqué dans certains formulaire, par exemple lorsque vous cliquez sur le texte associé à une case à cocher, cela coche la case ! Cela permet, notamment sur mobile, de grandement améliorer l'expérience utilisateur lors du remplissage du formulaire.
